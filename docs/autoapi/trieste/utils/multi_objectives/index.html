trieste.utils.multi_objectives¶

This module contains synthetic multi-objective functions, useful for experimentation.

Module Contents¶

class MultiObjectiveTestProblem[source]¶

Bases: abc.ABC

Base class for synthetic multi-objective test functions. Prepares the synthetic function and generates pareto optimal points. The latter can be used as a reference of performance measure of certain multi-objective optimization algorithms.

class VLMOP2[source]¶

Bases: MultiObjectiveTestProblem

The VLMOP2 problem, typically evaluated over \([-2, 2]^2\). The idea pareto fronts lies on -1/sqrt(2) - 1/sqrt(2) and x1=x2. See [VVL99] and :cite: fonseca1995multiobjective (the latter for discussion of pareto front property) for details.
