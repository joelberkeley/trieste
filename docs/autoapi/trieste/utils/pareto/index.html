trieste.utils.pareto¶

This module contains functions and classes for Pareto based multi-objective optimization.

Module Contents¶

non_dominated(observations: trieste.type.TensorType) → tuple[trieste.type.TensorType, trieste.type.TensorType][source]¶

Computes the non-dominated set for a set of data points. if there are duplicate point(s) in the non-dominated set, this function will return as it is without removing the duplicate.

Parameters

observations – set of points with shape [N,D]

Returns

tf.Tensor of the non-dominated set [P,D] and the degree of dominance [N], P is the number of points in pareto front dominances gives the number of dominating points for each data point

class Pareto(observations: trieste.type.TensorType, *, jitter: float = DEFAULTS.JITTER)[source]¶

A Pareto Construct a Pareto set. Stores a Pareto set and calculates the cell bounds covering the non-dominated region. The latter is needed for certain multiobjective acquisition functions.

For hypervolume-based multiobjective optimisation with n>2 objectives, this class implements branch and bound procedure algorithm. a divide and conquer method introduced in [CDD12].

Parameters

observations – The observations for all objectives, with shape [N, D].

Raises

ValueError (or InvalidArgumentError) – If observations has an invalid shape.

hypervolume_indicator(self, reference: trieste.type.TensorType) → trieste.type.TensorType[source]¶

Calculate the hypervolume indicator The hypervolume indicator is the volume of the dominated region.

Parameters

reference – a reference point to use, with shape [D]. Defines the upper bound of the hypervolume. Should be equal or bigger than the anti-ideal point of the Pareto set. For comparing results across runs, the same reference point must be used.

Returns

hypervolume indicator

Raises

  • ValueError (or tf.errors.InvalidArgumentError) – If reference has an invalid shape.

  • tf.errors.InvalidArgumentError – If reference is less than the anti-ideal point in any dimension.
