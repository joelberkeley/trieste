trieste.models.optimizer¶

This module contains model optimizers.

Module Contents¶

TrainingData[source]¶

Type alias for a batch, or batches, of training data.

DatasetTransformer[source]¶

Type alias for a function that converts a Dataset to batches of training data.

LossClosure[source]¶

Type alias for a loss closure, typically used in optimization.

OptimizeResult[source]¶

Optimization result. For scipy optimizer it is OptimizeResult. TensorFlow optimizer doesn’t return any result.

class Optimizer[source]¶

Optimizer for training models with all the training data at once.

optimizer :gpflow.optimizers.Scipy | tf.optimizers.Optimizer[source]¶

The underlying optimizer to use.

minimize_args :dict[str, Any][source]¶

The keyword arguments to pass to the minimize() method of the optimizer.

compile :bool = False[source]¶

If True, the optimization process will be compiled with tf.function().

optimize(self, model: tensorflow.Module, dataset: trieste.data.Dataset) → OptimizeResult[source]¶

Parameters

The return value of the optimizer’s minimize() method.

create_optimizer(optimizer: gpflow.optimizers.Scipy | tf.optimizers.Optimizer, optimizer_args: Dict[str, Any]) → Optimizer[source]¶

Generic function for creating a Optimizer wrapper from a specified optimizer and optimizer_args. The implementations depends on the type of the underlying optimizer.

Parameters

  • optimizer – The optimizer with which to train the model.

  • optimizer_args – The keyword arguments to pass to the optimizer wrapper..

Returns

The Optimizer wrapper.

create_loss_function(model, dataset: TrainingData, compile: bool = False) → LossClosure[source]¶

Parameters

  • compile – Whether to compile with tf.function().

Returns

The loss function.
